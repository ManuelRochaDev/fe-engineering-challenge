# Pokédex App

A modern Pokédex application built with React, TypeScript, and Vite. Browse, catch, and manage your collection of Pokémon with detailed stats and information.

## Features

- Browse all Pokémons with grid and table view modes
- View detailed stats and information for each Pokémon
- Catch Pokémons and manage your personal Pokédex collection
- Filter and sort Pokémon by various attributes
- Export your collection to CSV
- Share Pokémon with friends
- Consult your Pokédex, even in offline mode

## Tech Stack

- React + TypeScript
- Vite
- Vitest for unit tests

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Run tests
npm run test

# Build app
npm run build

# Run linter
npm run lint
```
